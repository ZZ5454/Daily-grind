Welcome to Large Bucks coffee webpage. This is a JavaScript project that takes a single webpage and rotates the content of the page based on the day of the week. The changes made to the site are changes to the site color as well as the name of the days coffee special its image and it's description this is done using switch blocks. The page also allows visitors to select days of the week and see the special for each day of the week and it's associated page design using QueryStrings
